== Description ==


To make the default "Ship to Different Address" checkbox unchecked in WooCommerce Checkout page, simply activate this plugin. To revert this, simply deactivate this plugin.  No settings page is neccessary for this plugin.


== Installation ==

To install, follow these steps:

1. Download and unzip the plugin

2. Upload the entire "ship-to-a-different-address-unchecked" directory to the /wp-content/plugins/

3. Activate the plugin through the Plugins menu in WordPress
